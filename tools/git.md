## 常用命令

#### git init

新建git仓库，会在当前文件夹多出一个.git文件夹

#### git status

查看当前文件的修改状态



#### git config

配置当前用户

配置用户名：git config --global user.name "your name"

配置邮箱：git config --global user.email "you email"



#### git add

git add -A

git add some_file



#### git commit

git commit -m "commit message"
